# Agent API 文档

## API 列表

### 1. control_character
- **功能**：控制角色的表情、动作、emoji、特效等  
- **调用时机**：  
  - 在扮演的角色将要发生情绪变化、做出动作、使用表情时  
  - 需向agent表达角色的心理想法或想要做出的动作，要求其生成相关表情、emoji和动作  

### 2. bg_changer  
- **功能**：切换背景  
- **调用时机**：  
  - 在用户提出前往某个地点时，命令其切换背景  

### 3. 记忆操作 API  
- **功能**：记录/查询/更新/删除角色的记忆  
  - `store_memory`：存储记忆  
  - `query_memory`：查询记忆  
  - `update_memory`：更新记忆  
  - `delete_memory`：删除记忆  
- **调用时机**：  
  - 需向agent提供信息的具体内容，要求其进行相关记忆操作  
  - 基本上每次对话都需要进行记忆操作  

## 记忆操作触发条件  
- 角色或对话者的感情变化/与对话者关系的变化  
- 重要情节推进或关键决策  
- 场景/时间线变更  
- 角色属性/状态更新  
- 用户特别要求记住的内容  
- 用户的相关信息
